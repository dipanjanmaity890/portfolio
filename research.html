FanPulse AI: A Browser-Native Framework for Passive Real-Time Sports Engagement Using LLM-Generated Push Notifications via Firebase Cloud Messaging

This paper presents FanPulse AI, a browser-native passive engagement framework that decouples sports information delivery from active user monitoring. The system integrates Google Gemini 1.5 Pro with Firebase Cloud Messaging (FCM) and the W3C Push API to deliver contextually rich, natural language match summaries as browser-level push notifications.
